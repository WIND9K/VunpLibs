1. Tổng quan về onuslibs
Mục	Nội dung
Tên	onuslibs (viết thường, không có dấu cách)
Ngôn ngữ	Python
Mục đích	Là thư viện chuẩn để gọi API nội bộ của hệ thống ONUS như: User, Transaction, Profile, Futures, KYC, Commission, v.v.
Đối tượng sử dụng	Dùng trong các dự án nội bộ như: OnusReport, fetch_commission_month, lock_outcome, process_txn
Cách cài đặt	Clone trực tiếp từ GitHub nội bộ:
pip uninstall onuslibs -y  
pip install git+https://github.com/WIND9K/OnusLibs.git

✅ 2. Cấu trúc của onuslibs

Thư viện được tổ chức theo cấu trúc module:

onuslibs/
│
├── api/                      # Chứa các client gọi API cụ thể
│   ├── user.py               # User API
│   ├── transaction.py        # Transaction API
│   ├── kyc.py                # KYC API
│   ├── commission.py         # Commission API
│   └── ...
│
├── core/                     # Chứa class và logic nền tảng
│   ├── api_client.py         # BaseClient: setup endpoint, token, retry
│   ├── config.py             # Load cấu hình như token, headers
│   └── exception.py          # Định nghĩa lỗi
│
├── utils/                    # Tiện ích chung
│   └── logger.py             # Logging chuẩn
│
├── __init__.py               # Entry point
└── pyproject.toml            # Định nghĩa package

✅ 3. Cách sử dụng onuslibs trong các dự án

Bước 1: Import class OnusLibsCredentials để truy cập các client API
(đã chuẩn hoá trong dự án như onusreport)

from config.onuslibs_adapter import OnusLibsCredentials

onus = OnusLibsCredentials()
user_api = onus.user_api
profile = user_api.get_profile(user_id)


Bước 2: Gọi các hàm tương ứng

# Lấy thông tin profile
profile = user_api.get_profile("1234567890123")

# Khóa tài khoản
user_api.freeze_user(user_id="1234", version_info="abcd", reason="vi phạm", frozen=True)

# Gọi Commission API
commission = onus.commission_api.get_monthly_commission(user_id, month="2025-10")

✅ 4. Tính năng chính của onuslibs
Tính năng	Mô tả
Tự động gắn token	Không cần tự thêm headers, đã config sẵn.
Tự động retry	Với các API lỗi tạm thời hoặc quá tải.
Chuẩn hoá URL, method, body	Các API đều gọi thống nhất qua BaseClient.
Ghi log lỗi	Ghi rõ endpoint, status code, response text.
Tích hợp dễ dàng	Có thể dùng trong mọi dự án không phụ thuộc cấu trúc.
✅ 5. Các API hiện được hỗ trợ
API	Mô tả
user_api	Lấy thông tin user, khóa tài khoản, cập nhật profile
transaction_api	Truy vấn giao dịch onchain, pro, exchange
kyc_api	Kiểm tra thông tin KYC
commission_api	Lấy commission theo tháng hoặc chi tiết
profile_api	Truy vấn và cập nhật profile theo user_id
futures_api	Lấy giao dịch futures
...	(Đang mở rộng thêm)
✅ 6. Tích hợp trong dự án OnusReport

Trong dự án OnusReport, onuslibs được tích hợp qua onuslibs_adapter.py:

from onuslibs.api import UserApi, TransactionApi, ProfileApi

class OnusLibsCredentials:
    def __init__(self):
        self.user_api = UserApi()
        self.transaction_api = TransactionApi()
        self.profile_api = ProfileApi()


Sau đó gọi trong Function/call_api.py như sau:

from config.onuslibs_adapter import OnusLibsCredentials

onus = OnusLibsCredentials()
user_data = onus.user_api.get_profile(user_id)
